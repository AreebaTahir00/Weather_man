# WeatherMan (Python) ☀️🌧️

A simple Python tool to analyze and visualize historical weather data from CSV files.

##  Features

- **Yearly Report**: Highest & lowest temperature with dates.
- **Monthly Averages**: Avg high, low, and humidity.
- **Daily Charts**: Console horizontal bar charts for max (🔴) and min (🔵) temperature per day.

##  Example Data (CSV format)
GST,Max TemperatureC,Min TemperatureC,Mean Humidity
2005-04-01,15,13,55
2005-04-02,20,10,60
